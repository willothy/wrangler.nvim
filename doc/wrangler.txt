Project-based mark manager built on SQLite

The goal of this plugin is to be a more minimal and customizeable equivalent to
ThePrimeagen/harpoon <https://github.com/ThePrimeagen/harpoon>, which hasn’t
been updated in a while and contains many features that I don’t use.


INSTALLATION                                           *wrangler-installation*

>lua
    require("lazy").setup({
      --- ...
      {
        "willothy/wrangler.nvim",
        config = true
      }
